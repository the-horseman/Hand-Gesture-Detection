Project Description

This is a web application that accepts hand gestures as input, detects them as one of 6 gestures, and then plays the game.

The dataset for the machine learning model was built by clicking many times on a single classifiable motion and then saving it for subsequent use in training the model. The information was then preprocessed. The aforementioned challenges were completed using OpenCV.

I trained the image recognition system successfully using a Convolutional Neural Network model that I designed.

The model was deployed on a web-based application, which displays the prediction of the User's gesture. It then feeds every collected frame to the model, which produces predictions. This prediction is then returned to the web application and shown alongside the frame that was provided to the model for prediction.
